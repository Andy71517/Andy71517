### IdolTracker — News Aggregation Frontend

Frontend for a team-built idol news aggregation MVP.

I led much of the frontend implementation for a React/TypeScript application that presents processed article data through a coverage feed, artist/group/source views, search, filtering, pagination, and a two-pane master-detail reading experience.

**Highlights**

* Built the main coverage feed UI for article discovery and reading
* Implemented route-based views for artists, groups, and sources
* Added search and filters for sentiment, language, and country
* Used TanStack Query for API-backed article fetching and cache behavior
* Built master-detail article selection with loading, empty, and error states
* Persisted filters locally to improve user experience across sessions

**Tech and areas**

React, TypeScript, Vite, TanStack Query, React Router, Zustand, Tailwind CSS, API integration, frontend state management.
